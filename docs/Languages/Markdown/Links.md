# Links

Links can be both external URLs and internal URLs.

## Inline links

Links can be written inline.

Syntax:

```markdown
[The text](the link)

[Google](www.google.com)
```

## Reference links

Syntax:

```markdown
[The text][id]

[Google][id1]
[id1]: www.google.com "Google"
```
