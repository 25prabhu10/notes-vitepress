# Lists

## Unordered lists

You can use `*`, `+`, and `-`.

Syntax:

```markdown
- Candy
- Gum
- Soda

* Candy
* Gum
* Soda

- Candy
- Gum
- Soda
```

## Ordered lists use numbers

Like...

1. Candy
2. Gum
3. Soda

## For multi-paragraph list items use 4 spaces or 1 tab at the end of first line

Like...

- A list item. with multi-paragraphs.
- Another item.  
  with 4 spaces.
